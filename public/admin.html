📡 How to use

Add this header to your HTTP requests:

Available endpoints:

GET /api/subscriptions → subscriptions:read GET /api/subscriptions/:id → subscriptions:read POST /api/subscriptions/:id/cancel → subscriptions:write POST /api/subscriptions/:id/reactivate → subscriptions:write GET /api/analytics/overview → analytics:read GET /api/products → products:read
